# Call-center-Performance-Dashboard-using-Excel
📊 Call Center Analytics Dashboard (Excel)

📌 Overview

This project presents an interactive Call Center Dashboard built using Microsoft Excel to analyze customer support performance. The dashboard provides key insights into call volume, response time, agent performance, and customer satisfaction.

---

🎯 Objective

- Monitor call center performance metrics
- Analyze agent efficiency and productivity
- Identify trends and areas for improvement
- Support data-driven decision-making

---

🧠 Tools & Technologies

- Microsoft Excel
  - Pivot Tables
  - Pivot Charts
  - Slicers
  - Conditional Formatting

---

📂 Dataset

- Call center dataset containing:
  - Call ID
  - Agent Name
  - Call Duration
  - Response Time
  - Call Status (Resolved / Unresolved)
  - Customer Satisfaction Score

---

⚙️ Dashboard Features

- 📞 Total Calls Handled
- ⏱ Average Response Time
- ✅ Resolution Rate
- ⭐ Customer Satisfaction Score
- 👨‍💼 Agent-wise Performance Analysis
- 📅 Time-based Trends (Daily/Monthly)

---

📊 Key Insights

- Identified peak call hours and workload distribution
- Highlighted top-performing and underperforming agents
- Found correlation between response time and customer satisfaction

---

🚀 Future Improvements
